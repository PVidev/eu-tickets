# EU Tickets - Система за търсене на транспортни билети

## Описание
EU Tickets е иновативна платформа, която революционизира начина, по който пътниците планират своите европейски приключения. Като водещ доставчик на пътнически решения, ние сме установили стратегически партньорства с водещите транспортни компании в Европа, за да предоставим най-изчерпателните и конкурентни пътнически опции.

## Нашата мисия
Нашата цел е да направим пътуването в Европа по-лесно, по-достъпно и по-приятно за всеки. Чрез нашите стратегически партньорства и иновативна платформа, ние:
- Предоставяме достъп до най-добрите маршрути и сделки
- Гарантираме най-добрата стойност за вашите пари
- Осигуряваме 24/7 поддръжка
- Предлагаме гъвкави опции за пътуване

## Функционалности
- Търсене на транспортни услуги по държава и тип транспорт
- Сравнение на различни транспортни оператори
- Детайлна информация за всеки оператор
- Интерактивен интерфейс с разгъващи се детайли
- Responsive дизайн за всички устройства
- Система за контакт с различни отдели
- Защита срещу ботове с динамична captcha система

## Функционални компоненти

### Търсене
- Избор на държава от падащо меню
- Избор на тип транспорт (влак, автобус, ферибот)
- Бутон за търсене, който показва резултатите

### Резултати
- Карти с информация за всеки оператор
- Бутони за:
  - Посещение на уебсайта
  - Показване на базова информация
  - Показване на детайлна информация

### Контактна форма
- Валидация на всички полета
- Динамична captcha система
- Различни отдели с различни имейл адреси
- Ограничение на дължината на съобщението (60-1500 символа)
- AJAX изпращане на форми
- Логване на всички съобщения

### Детайлна информация
Разгъваща се секция, съдържаща:
- Описание на оператора
- Функции и услуги
- Правила за резервация
- Предимства
- Контактна информация
- Правила за възстановяване

## Инсталация и стартиране
1. Клонирайте репозиторито
2. Конфигурирайте PHP среда
3. Настройте имейл сървъра
4. Уверете се, че Apache сървърът има активиран mod_rewrite модул
5. Конфигурирайте SSL сертификат за HTTPS
6. Отворете `index.html` във вашия браузър

## Apache конфигурация (.htaccess)
Файлът .htaccess съдържа важни настройки за сигурност и оптимизация:

### URL оптимизация
- Премахване на .html разширенията от URL адресите
- Пренасочване от www към non-www версия
- Пренасочване от HTTP към HTTPS

### Сигурност
- Защита на системни файлове и директории
- Защита на JSON и PHP файлове
- Защита от XSS атаки
- Защита от MIME-type sniffing
- Защита от clickjacking
- Контрол на referrer информацията

### Оптимизация
- Кеширане на статични файлове
- Компресия на файлове
- Обработка на грешки (404, 403, 500)

### Изисквания за сървъра
- Apache сървър с mod_rewrite модул
- SSL сертификат за HTTPS
- PHP 7.4 или по-нова версия
- mod_headers модул
- mod_expires модул
- mod_deflate модул
